# Assets Directory

This directory contains static assets for the HisaabKitab application.

## Required Assets

- `icon.png` - Application icon (PNG format, 256x256 recommended)
- `icon.ico` - Windows icon file (for Windows builds)
- `logo.png` - Application logo
- `favicon.ico` - Browser favicon

## Adding Assets

Place your logo, icons, and other static assets in this directory. The application will reference these files from here.

## Notes

- Icon files should be high resolution for best display quality
- Ensure PNG transparency where needed
- ICO files are required for Windows builds
